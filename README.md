# Entropy Tree Map

This Python package generates interactive treemap graphs, containing entropy information.

Low-entropy areas can then be targeted by your favorite compression tool.

### Dependencies

- Python 3
- Python packages:
  - `tqdm`
  - `plotly`
  - `jinja2`
  - `pywebview`
